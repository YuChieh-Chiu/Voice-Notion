# 🚀 Siri 捷徑快速設定指南 (試用者專用)

> [!TIP]
> **本文件專為使用 [展示網頁](https://voice-notion.jacktoholiday.uk/demo) 的使用者設計。**
> 只要您可以從網頁端複製到「封裝後的設定 JSON」，請按照以下步驟快速將其匯入 iOS 捷徑。

---

## 步驟 1：網頁端準備
1. 在展示網頁輸入您的 Gemini API Key 與 Notion Token。
2. 點擊「封裝捷徑設定」按鈕。
3. **複製**產生的那一串包含大括號 `{...}` 的 JSON 代碼。

## 步驟 2：iOS 捷徑建立
1. 開啟 iPhone 上的「**捷徑 (Shortcuts)**」App，點擊右上角「+」建立新捷徑。
2. **加入動作**：搜尋「**錄音 (Record Audio)**」。
3. **加入動作**：搜尋「**取得 URL 內容 (Get Contents of URL)**」。
   - **URL**：輸入展示網頁中提供的 API 端點。
   - 點擊展開箭頭 (顯示更多)：
     - **方法 (Method)**：選擇 `POST`。
     - **標題 (Headers)**：
       1. 點擊「加入新標題」。
       2. **鍵 (Key)**：輸入 `X-Voice-Notion-Config`。
       3. **值 (Value)**：**貼入**剛才從網頁複製的那一串 JSON 代碼。
     - **要求內文 (Request Body)**：選擇「**檔案 (File)**」。
     - **檔案 (File)**：選擇變數「**錄製的音訊**」。
4. **加入動作**：搜尋「**顯示通知**」並連結 `URL 內容`。
5. 將捷徑命名為「**記筆記**」或任何您喜歡的 Siri 喚醒詞。

---

## ⚡ 喚醒方式
對著 iPhone 說：「**嘿 Siri，記筆記**」，錄音完成後您的筆記便會自動飛往 Notion！
