# 🌈 Voice-Notion 快速試用指南 (Demo Guide)

歡迎來到 Voice-Notion 展示模式！您可以無需部署任何程式碼，僅憑自己的 API Key 即可體驗 AI 語音轉 Notion 的服務功能。

---

## 快速開始三部曲

### 1. 取得 API 金鑰
- **Google Gemini**: 前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 免費申請。
- **Notion**: 前往 [My Integrations](https://www.notion.so/my-integrations) 建立整合，並**授權**給您的目標頁面。

### 2. 使用展示網頁設定
- 開啟展示頁面：[Demo Page](https://voice-notion.jacktoholiday.uk/demo)
- 填入金鑰，並產生「捷徑配置 JSON」。

### 3. 設定 iOS 捷徑
- 參考 [Siri 捷徑快速設定指南 (Demo 版)](./SIRI_INTEGRATION_DEMO.md) 將配置貼入您的 iPhone。

---

## ⚠️ 試用限制與聲明
- **速率限制**：因為部署資源有限，展示模式下的 IP 速率限制為每小時 3 次請求。
- **數據隱私**：您的 API Key 僅儲存在您的手機與本地瀏覽器中，伺服器不會留存。
